# NikaGeospatial Homebrew Tap

This is the official Homebrew tap for NikaGeospatial tools.

## Installation

```bash
brew tap NikaGeospatial/geoengine
brew install geoengine
```

## Available Formulae

- **geoengine** - Docker-based isolated runtime manager for geospatial workloads

## Documentation

For more information, visit the [GeoEngine repository](https://github.com/NikaGeospatial/geoengine).
